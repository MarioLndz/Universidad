Detailed Description

T.D.A. MaxQueue.

Una instancia del contenedor MaxQueue nos permite usar una Cola (Queue) en la que se tendrá registro en todo momento del mayor elemento almacenado en la cola

Para poder usar el TDA MaxQueue se debe incluir el fichero

#include <maxqueue.h>

Inserta un nuevo elemento al end de la cola.
Los elementos anteriores de la cola verán modificado su máximo registrado. Esto se debe a que el nuevo elemento X insertado será el último en salir de la cola, por lo que si éste es el valor máximo se deberá ver reflejado en los elementos por encima de él.
